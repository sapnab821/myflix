myFlix

To build serverside component of a movies application. It will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update personal information and creat a list of their favorite movies

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None None A JSON object of all movies
Return data (description, genre, director, image URL, whether its featured or not) about a single movie by title to the user /movies/:title GET :title None A JSON object of movie description based on title
Return data about a genre (description) /movies/:genre/:genreName GET :genreName None A JSON object a the description of a specific genre
Return data about a director (bio, birth year, death year) by name /movies/director/:directorName GET :directorName None A JSON object of the description of director
Allow new users to register /users Post None A JSON object of user information ex. { "Username": "cloud45", "Password": "Mountain34", "Email": "gems29@gmail.com", "Birthday": "1989-03-04" } A JSON object of user information ex. { "_id": "64b947c90f28f0ee8a2fab7e", "Username": "cloud45", "Password": "Mountain34", "Email": "gems29@gmail.com", "Birthday": "1989-03-04T00:00:00.000Z" }
Allow users to update their user info /users/:id PUT :id A JSON object of updated user information ex. { "Password": "po54nd" } A JSON object of of the users complete information ex. { "_id": "64b9429ee4e96a4a8115647c", "Username": "Sarah98", "Password": "po54nd", "Email": "sarahv@yahoo.com", "Birthday": "1982-01-25T00:00:00.000Z" },
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:id/:movieTitle POST :id, :movieTitle None JSON text of all data on user including update movies list ex. { "_id": "64c11d98392cde19afcee2e9", "Username": "Billy1", "Password": "vody", "Email": "sam@gmail.com", "Birthday": "2001-09-30T00:00:00.000Z", "FavoriteMovies": [ "64b83ab4d07cb110a4b60d2b" ], "__v": 0 }
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:id/:movieTitle DELETE :id, :movieTitle None A text confirming the movie has been removed from user's favorites
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later). /users/:id DELETE :id None A text confirming the user has been removed
